  # Bagy

  ## **Plataforma de e-commerce integrada via API**

  Integração com a plataforma de e-commerce **Bagy**, facilitando ainda mais a gestão do seu negócio. Com essa novidade, é possível importar automaticamente os pedidos e os dados dos clientes cadastrados na Bagy para o seu CRM. Essa funcionalidade elimina o trabalho manual, economiza tempo e garante que suas informações estejam sempre atualizada.
